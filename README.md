Standalone python file that creates `bootanimation.zip` from a gif and automatically set's **framerate**.

# Depends
- ffmpeg

# Code

## Extract frames
Extract frames from gif file making sure that the bg of any transparent gif is black
Generated by AI (gemini)
```bash
# Extract frames from gif:
ffmpeg -i input.gif -filter_complex "[0:v]split[fg][bg];[bg]format=pix_fmts=rgb24,geq=r='0':g='0':b='0'[bg_black];[bg_black][fg]overlay=shortest=1[out]" -map "[out]" -q:v 2 -y bootanimation/part0/out_%04d.jpg
```

## Scale frames
Scale frame while maintaing aspect ratio of frame and filling the neccesary space with black color also uses `pad` option of `ffmpeg` to align the frame.
```bash
# Scale frames to correct dimensions:
for f in bootanimation/part0/*.jpg; do ffmpeg -i "$f" -loglevel quiet -vf "scale=800:1280:force_original_aspect_ratio=decrease,pad=800:1280:(ow-iw)/2:(oh-ih)/2:black" "bootanimation/part0/${f#out_}"; done;
```

## Final Command
Combined version of above.
```bash
# Final command
ffmpeg -i input.gif -loglevel fatal -filter_complex "[0:v]split[fg][bg];[bg]format=pix_fmts=rgb24,geq=r='0':g='0':b='0'[bg_black];[bg_black][fg]overlay=shortest=1[out]" -map "[out]" -q:v 2 -y bootanimation/part0/out_%04d.jpg && for f in bootanimation/part0/*.jpg; do filename=$(basename "$f"); ffmpeg -i "$f" -loglevel fatal -vf "scale=800:1280:force_original_aspect_ratio=decrease,pad=800:1280:(ow-iw)/2:(oh-ih)/2:black" -y "bootanimation/part0/${filename#out_}" && rm "$f" && echo "${filename#out_}" Scaled!; done;
```

## Zip Folder
Zip the files with **0** decompression level!
```bash
# Zip files
cd bootanimation && zip -0r ../bootanimation.zip .
```
